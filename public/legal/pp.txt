# Privacy Policy

Last Updated: January 16, 2025

- Educational information
- Account preferences
- Country and language settings

### 1.2 Usage Data
- Login activity and device information
- Feature usage patterns
- Document processing history
- Study session data

### 1.3 User Content
- Uploaded documents
- Generated study materials
- Audio recordings
- User annotations

### 2.1 Service Provision
- Process and store documents
- Generate study materials
- Manage user accounts
- Provide customer support

### 2.2 Service Improvement
- Analyze feature usage
- Improve algorithms
- Debug issues
- Enhance user experience

## 3. Data Storage and Security

### 3.1 Data Storage
- Industry-standard encryption
- Secure cloud storage
- Regular security audits
- Backup procedures

### 3.2 Data Retention
- Account data retained while active
- Content deleted 30 days after account closure
- Usage data anonymized after 12 months

### 4.1 Access and Control
Users can:
- Access their personal data
- Request data correction
- Download their content
- Delete their account

### 4.2 Data Portability
- Export study materials
- Transfer content to other services
- Receive data in common formats

### 5.1 Service Providers
We may share data with:

### 5.2 Data Processing Agreements

- Regular security assessments

We use cookies for:
